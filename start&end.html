Start & End G-Code

Start G-code는 출력 전에 프린터를 준비시키는 과정으로, 예열·홈 이동·프라임 라인 등을 자동으로 수행합니다.
End G-code는 출력 후 장비를 안전하게 정리하며, 노즐·베드 온도 감소와 노즐 이동, 모터 해제 등을 담당합니다.

Start G-code

출력 전 준비 과정과 출력 후 마무리 과정으로 구성되며, 예열 → 홈 이동 → 압출 라인 → 종료 동작 순으로 수행됩니다. 

; ====== Start G-code ======
M104 S{temperature_nozzle}
M140 S{temperature_bed}
M190 S{temperature_bed}
M109 S{temperature_nozzle}

G28
G92 E0
G1 Z2.0 F3000
G1 X2.0 Y10 F5000
G1 Z0.28 F1200
G1 X200.0 E15 F1500
G1 X220.0 F5000
G92 E0

End G-Code

출력 완료 후 프린터를 안전하게 정리하는 코드입니다.
노즐 냉각, 베드 하강, 압출기 후진 등이 포함됩니다. 

; ====== End G-code ======
G91                     ; 상대 좌표 모드
G1 E-2 F2400            ; 필라멘트 후진
G1 Z10 F3000            ; 노즐 상승
G90                     ; 절대 좌표 복귀
G1 X0 Y220 F4000        ; 노즐 대기 위치 이동

M104 S0                 ; 노즐 끄기
M140 S0                 ; 베드 끄기
M107                    ; 팬 정지
M84                     ; 모터 전원 해제
; ====== End G-code End ======
