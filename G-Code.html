G-Code

G-code는 3D 프린터가 움직임과 온도 같은 작업을 수행할 수 있도록 지시하는 표준 명령어 형식입니다.
슬라이서에서 자동으로 생성된 G-code를 사용하면 프린터가 출력 과정을 정확하게 따라가며 안정적인 결과물을 얻을 수 있습니다.

📘 G-code & M-code 명령어 정리 (주요 명령 위주)

3D 프린터가 이해하는 G-code는 이동(G코드)과 설정(M코드)으로 나뉘며, 아래 표는 FDM 3D 프린터에서 많이 사용하는 명령들을 정리한 것입니다.

💡 Tip. 펌웨어(예: Marlin, Klipper, RepRap)에 따라 지원하는 코드가 다를 수 있으니, 실제 사용 전에는 항상 해당 펌웨어 문서를 확인하세요.
명령어 분류 역할 / 설명 예시
G0 이동 빠른 공회전 이동(비압출 이동). 보통 G1과 동일하게 처리되기도 함. G0 X100 Y120 F6000
G1 이동 직선 보간 이동. 압출(E) 값을 함께 써서 출력 경로를 지정. G1 X10 Y20 Z0.3 E0.5 F1200
G2 이동 시계 방향 원호(원) 보간 이동. 일부 펌웨어에서만 지원. G2 X50 Y50 I10 J0
G3 이동 반시계 방향 원호 보간 이동. G3 X50 Y50 I-10 J0
G4 시간 Dwell, 잠시 멈춤. P 또는 S로 시간 지정. G4 P5000 (5초 대기)
G20 단위 단위를 인치로 설정. (일반 FDM에서는 거의 사용 안 함) G20
G21 단위 단위를 mm로 설정. 대부분의 슬라이서 기본값. G21
G28 홈 모든 축 또는 지정 축을 홈 위치(원점)로 이동. G28, G28 X Y
G90 좌표 절대 좌표 모드로 변경 (X/Y/Z는 0 기준 절대 위치). G90
G91 좌표 상대 좌표 모드 (현재 위치를 기준으로 이동량 지정). G91
G92 좌표 현재 위치를 임의의 값으로 재설정. 주로 E값 리셋에 사용. G92 E0
M82 압출 압출량을 절대 모드로 설정. 대부분의 상용 슬라이서 기본. M82
M83 압출 압출량을 상대 모드로 설정. M83
M104 온도 노즐 온도 설정(기다리지 않음). 다음 동작과 동시에 예열만 시작. M104 S200
M109 온도 노즐 온도 설정 후, 목표 온도에 도달할 때까지 대기. M109 S200
M140 온도 베드 온도 설정(기다리지 않음). M140 S60
M190 온도 베드 온도 설정 후, 도달할 때까지 대기. M190 S60
M106 쿨링 파트 쿨링 팬 속도 설정. S0~255 (또는 0~100%). M106 S255
M107 쿨링 파트 쿨링 팬 끄기. M107
M17 모터 스테핑 모터(모든 축) 활성화. (펌웨어에 따라 자동 온/오프) M17
M18 / M84 모터 스테핑 모터 비활성화(토크 해제). 축을 손으로 움직일 수 있게 됨. M84
M0 일시정지 프로그램 일시 정지. 일부 펌웨어에서 LCD에 메시지 표시 가능. M0 무언가 교체하세요
M1 일시정지 조건부 일시정지 (펌웨어에 따라 동작 유사). M1
M92 보정 각 축의 steps/mm 값을 설정. X/Y/Z/E 스텝 보정. M92 X80 Y80 Z400 E420
M500 보정 현재 설정값(스텝수, PID 등)을 EEPROM에 저장. M500
M501 보정 EEPROM에 저장된 설정값 불러오기. M501
M502 보정 펌웨어 기본값으로 리셋(EEPROM 값 초기화). M502
M115 정보 펌웨어 버전 및 보드 정보 요청. 터미널에 세부 정보 출력. M115
M114 정보 현재 좌표/위치 정보 출력. M114
M119 정보 엔드스톱 상태 확인(눌림/열림 상태). 센서 점검용. M119
M303 PID 노즐 또는 베드의 PID 자동 튜닝. M303 E0 S200 C8
M220 속도 전체 프린트 속도를 비율로 조정(Feed rate override). M220 S120 (120%)
M221 압출 압출 비율 조정(Flow rate override). M221 S95 (95%)
M600 필라멘트 필라멘트 교체 또는 색상 변경 루틴 실행. 펌웨어 지원 필요. M600
M420 레벨링 메시 베드 레벨링 사용/해제 및 상태 설정. M420 S1 (사용)
M204 가속도 프린트/이동 가속도 설정. M204 P1000 T1500
M205 저크/제한 Jerk, Junction Deviation 등 동작 한계 값 설정. M205 X10 Y10 Z0.4
M900 압출 Linear Advance(K값) 설정. 고속에서 압출 보정. M900 K0.05
M82 + G92 E0 압출 압출 모드/카운터 초기화 조합. 시작 G-code에서 자주 사용. M82
G92 E0
M104 S0 온도 노즐 히터 끄기 (출력 종료 후). M104 S0
M140 S0 온도 베드 히터 끄기. M140 S0
G1 X0 Y220 이동 출력물 꺼내기 좋게 헤드를 옆으로 이동(엔드 G-code 예시). G1 X0 Y220 F6000
M84 모터 출력 종료 후 모터 전류 차단(축 프리). M84
